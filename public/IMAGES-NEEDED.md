# Required Images for SEO

Place these images in the `/public` folder for complete SEO optimization.

## üé® Required Images

### 1. Open Graph Image (Social Sharing)
**Filename**: `og-image.jpg`
**Size**: 1200 x 630 pixels
**Format**: JPG or PNG
**Purpose**: Displayed when your site is shared on Facebook, LinkedIn, Twitter, etc.
**Content**: Should include:
- Your logo/brand name
- Tagline or key message
- Professional water damage restoration imagery
- Contact number (optional)

**Design Tips**:
- Keep text large and readable
- Use high contrast
- Avoid placing important content near edges
- Test how it looks on different platforms

---

### 2. PWA Icons

#### Small Icon
**Filename**: `icon-192.png`
**Size**: 192 x 192 pixels
**Format**: PNG with transparency
**Purpose**: Progressive Web App icon (Android)

#### Large Icon
**Filename**: `icon-512.png`
**Size**: 512 x 512 pixels
**Format**: PNG with transparency
**Purpose**: Progressive Web App icon (Android, splash screens)

**Design Tips**:
- Simple, recognizable logo
- Works well at small sizes
- Transparent background
- Centered design with padding

---

### 3. Favicon
**Filename**: `favicon.ico`
**Size**: 32 x 32 pixels (multi-size ICO recommended)
**Format**: ICO file
**Purpose**: Browser tab icon

**How to Create**:
1. Design a 512x512 PNG
2. Use online converter: https://favicon.io/favicon-converter/
3. Download the ICO file

---

### 4. Apple Touch Icon
**Filename**: `apple-touch-icon.png`
**Size**: 180 x 180 pixels
**Format**: PNG
**Purpose**: iOS home screen icon

**Design Tips**:
- iOS automatically adds rounded corners
- Don't add your own rounded corners
- Use solid background (no transparency)

---

### 5. Logo
**Filename**: `logo.png`
**Size**: Flexible (recommend 500 x 200 pixels)
**Format**: PNG with transparency
**Purpose**: Used in schema markup and potentially on site

## üîç Testing Your Images

After adding images, test them:

1. **Open Graph**:
   - Facebook: https://developers.facebook.com/tools/debug/
   - LinkedIn: https://www.linkedin.com/post-inspector/
   - Twitter: https://cards-dev.twitter.com/validator

2. **PWA Icons**:
   - Use Chrome DevTools > Application > Manifest
   - Test on actual mobile device

3. **Favicon**:
   - Check browser tab after deployment
   - Test in different browsers

---

## üí° Pro Tips

1. **Optimize file sizes**: Keep images under 200KB when possible
2. **Use descriptive filenames**: Helps with SEO
3. **Maintain aspect ratios**: Don't stretch or distort images
4. **Test on multiple devices**: Ensure images look good everywhere
5. **Keep source files**: Save high-res versions for future updates
